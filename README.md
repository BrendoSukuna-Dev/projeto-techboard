## 📋 Sobre o projeto
<p>O TechBoard é uma Landing Page moderna desenvolvida para apresentar uma solução de monitoramento de aplicações. O foco do projeto é oferecer uma interface limpa, responsiva e visualmente impactante,
  utilizando uma paleta de cores voltada para o modo escuro (Dark Mode) com detalhes em roxo vibrante.</p>

## ✦ Principais funcionalidades:

<p>➥ Design Moderno: Uso da fonte Poppins para uma leitura clara e visual contemporâneo.</p>
<p>➥ Seção Hero Impactante: Título chamativo com destaque em cores contrastantes e descrição clara do serviço.</p>
<p>➥ Responsividade Total: Layout adaptável para dispositivos móveis (375px), tablets (768px) e desktops, garantindo que o conteúdo seja exibido corretamente em qualquer tela.</p>
<p>➥ Chamada para Ação (CTA): Botão estilizado com efeitos de hover para incentivar a interação do usuário.</p>
